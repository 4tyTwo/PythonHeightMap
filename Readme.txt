Алгоритм diamond square:https://ru.wikipedia.org/wiki/%D0%90%D0%BB%D0%B3%D0%BE%D1%80%D0%B8%D1%82%D0%BC_Diamond-Square


На данный момент алгоритм выстроен так, чтобы сгладить края карты и погрузить их под воду.
Можно заметить небольшие артефакты возле границ, скорее всего их можно будет убрать при помощи возведения крайних элементов матрицы в степень.
